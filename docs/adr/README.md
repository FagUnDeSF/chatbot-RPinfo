Trilha publica de decisoes arquiteturais do projeto em formato legivel fora da
sala do Tech Lead.

## Fonte tecnica

Os ADRs vivos continuam sob autoridade do Tech Lead na sala operacional local.
Este diretorio publica espelhos sanitizados para portfolio, sem expor artefatos
operacionais internos.

## Regra de uso

1. Publique aqui apenas ADR aprovado ou espelho publico aprovado de ADR da sala TL.
2. Preserve numeracao com quatro digitos.
3. Registre status `proposto`, `aprovado`, `superseded` ou `rejeitado`.
4. Nao edite conteudo tecnico sem decisao do TL.
